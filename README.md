# Detecting Strategic Deception in Reputation Games Using Inverse Reinforcement Learning

This repository contains the Python implementation for my undergraduate thesis, which investigates the application of Non-Cooperative Inverse Reinforcement Learning (N-CIRL) methods to detect strategic deception in repeated game-theoretic settings. 

The models simulate a competition policy game where agents attempt to build and exploit reputation through deceptive strategies. By applying a Point Based Value Iteration algorithm and linear programming, this project reconstructs the hidden reward functions of these agents to differentiate between authentic cooperation and strategic manipulation.
